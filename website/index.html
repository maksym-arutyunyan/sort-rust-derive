Rust Derive Traits Sorter

What:

Quickly and easily organize your Rust derive traits

Why:

Sorting derive traits ensures consistency in your Rust projects. It makes it easier for developers to scan and understand the code by keeping derives in a predictable order.

Although cargo fmt doesn't support sorting derive traits due to some rare edge cases, this tool can help you keep consistent trait order in most situations.

Sorting Methods:
  • Canonical: Standard Rust traits come first, and the rest follow in alphabetical order.
  • Alphabetical: All traits are listed alphabetically.
